New View Menu

Designed for Apple (Claris) in 2022

This project was designed for Claris Studio, a cloud-based data collection and management platform created by Claris, a subsidiary of Apple, Inc. One of my first tasks as a new designer on the team was to perform a usability test on myself to complete a task. In this case, create a web form that I would distribute to the team. Long story short, I spent three days working in the wrong feature (referred to as a "view" in Claris Studio) because there was not enough information provided to me to perform the correct action. This project was an exploration to provide more information to new and returning users so they can make the right choice the first time.

The Problem: First time users do not have enough information to make well-educated decisions on which "view" to create for a given task.

When introduced to Claris Studio for the first time, you're introduced to new concepts and terminology without education. So when first attempted to complete a given task, there is high likelihood of making a short to long-term mistake. The goal of this project was to explore alternative "new view" menus that could provide more education to the users about the different view types and when to use each one.
